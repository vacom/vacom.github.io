What is Dina Framework?

Dina is a framework for developing web applications and sites. Dina uses modern technologies and updated to make his philosophy more simple and stable.

Dina as a structure like the others have the basic elements included with this diversity and become a complete tool for any designer and programmer begin immediately to develop their applications. Will contain basic HTML and CSS templates for typography, forms, buttons, graphics, navigation and other user interface components and to assist their interaction and fluidity extinctions javascript.

Many of these components are not included in the initial package and this because, and here is the whole essence to stand out from the others. Many of these components I will not develop because the goal is to be an open framework for the developer community. So many of the features have a touch of other programmers interested.

To make Dina particular its evolution will be developed as cakes and these cakes have ingredients, these are the components.
